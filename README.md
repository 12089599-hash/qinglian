# 青岚洞府

一个可直接托管的网页挂机修仙小游戏。

## 当前玩法

- 按分钟吐纳灵气和离线收益，早期保留推进感，中后期节奏明显放缓
- 36 个小境界：炼气、筑基各九层，金丹九转，元婴九变
- 破境天机、心魔压力，境界曲线已按至少一周冲击元婴的长期挂机节奏重调
- 洞府升级：蒲团、灵田、剑阵
- 丹方系统：聚气丹、清心丹、护脉丹
- 炼丹炉升级：缩短炼丹时间并解锁丹方
- 历练任务和多路线秘境战斗，产出妖核与法器
- 历练任务可设置自动重复
- 历练会触发奇遇：灵泉、药圃、游商、妖兽伏击、残剑秘匣等
- 奇遇可能掉落具名战利品：青锋剑、玄木护符、云纹法袍
- 战利品可穿戴到武器、护符、法袍栏，提供额外道威、灵息、破境天机或秘境减险
- 历练按地图分层：青岚山、灵草谷、雾隐秘境、古剑冢、魔气裂隙、上古遗迹
- 高阶地图按境界解锁，显示卦象、主产物和稀有奖励进度
- 每张地图拥有探索进度、地图声望、熟练度和首领挑战
- 镇压地图首领可获得一次性永久奖励和炼器精魄
- 主线章节：青岚初启、筑基问道、金丹试炼，每章包含阶段目标和篇章奖励
- 后期主线延伸到裂隙镇守、元婴开宗
- 页面会显示当前最推荐的下一步行动
- 修士档案可展开查看道威、灵息、破境天机、护体玄光和山门气运的来源
- 修仙目标可领取一次性奖励，完成整章后获得永久成长加成
- 完成 3 个新手目标后解锁进度型日常任务
- 坊市可购买材料、装备和阵法资源
- 炼丹需要时间，服丹后获得限时灵息加速
- 装备栏和阵法栏可升级，影响道威、灵息和破境天机
- 装备可用法器和灵石淬炼品质：凡品、下品、中品、上品、极品
- 装备卡片可展开查看器象、等级、品质、具体属性和词条效果，词条包含剑意、破阵、灵脉、凝神、云步、护体等
- 多余战利品可分解为炼器精魄，炼器精魄可强化具名战利品；跨入灵阶、玄阶、地阶后材料和加成曲线都会上升
- 法宝系统：本命青印、养剑葫、聚灵灯，可长期温养并影响破境天机、道威、灵息和行游劫象
- 灵兽系统：云纹灵狐、雷纹幼虎，可培养并提供灵息、灵草、道威或护山收益
- 功法流派：剑修、丹修、阵修三条长期路线，可形成主修方向
- 宗门系统：炼气三层后招募具名弟子，弟子拥有资质、根骨、悟性、灵根和委托经验
- 宗门声望会提升宗门品阶，从外门草创成长到洞天雏形，解锁更多弟子容量和委托效率
- 历练可触发可选择的机缘事件，例如剑冢回响、灵泉分流、魔念低语
- 剑修凝练道威和秘境稳定，丹修提高炼丹与服丹收益，阵修提高灵息和挂机成长
- 升级梯度分为凡阶、灵阶、玄阶、地阶，境界越高可升级上限越高
- 历练和首领使用卦象、劫象、宜备提示展示战前判断
- 中后期升级成本递增，洞府、装备、阵法分别消耗不同核心资源
- 突破前可稳固根基，提高把握并缓冲心魔
- 功能面板使用标签页切换，手机端使用底部导航

## 本地打开

直接用浏览器打开 `index.html`。

## 发布到 GitHub Pages

1. 在 GitHub 新建一个公开仓库，例如 `qinglan-dongfu`。
2. 如果用 GitHub 网页上传，覆盖根目录的 `index.html`、`styles.css`、`browserGame.js`、`README.md`。
3. 进入仓库 `Settings` -> `Pages`。
4. `Build and deployment` 选择 `Deploy from a branch`。
5. `Branch` 选择 `main`，目录选择 `/root`，保存。
6. 等待部署完成后，GitHub 会给出一个可分享的网址。

## 发布到 Netlify 或 Vercel

把整个文件夹拖到 Netlify，或导入到 Vercel。构建命令留空，发布目录选择项目根目录。

## 文件说明

- `index.html`: 游戏入口页面
- `styles.css`: 页面样式
- `browserGame.js`: GitHub Pages 直接运行的游戏脚本
- `src/browserGame.js`: 浏览器直接运行的游戏版本
- `src/gameCore.mjs`: 可测试的核心规则
- `tests/game.test.mjs`: 核心规则测试
